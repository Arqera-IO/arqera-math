# arqera-math

Mathematical foundations extracted from ARQERA's engine for reuse across projects.

## Modules

- **bayesian** — Beta-binomial trust model (BayesianTrustService)
- **graph_analysis** — PageRank, centrality, clustering (GraphAnalysisService)
- **information_theory** — Shannon entropy, pruning (InformationTheoryService)
- **constants** — Tuned mathematical constants registry

## Install

```bash
pip install -e .
```
